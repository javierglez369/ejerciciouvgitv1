# Mi Proyecto de Git

## Descripción

Este es un proyecto simple creado para aprender y practicar los conceptos fundamentales de Git y versionado de código. El proyecto contiene una página web básica con HTML y CSS.

## Contenido

- **index.html** - Página principal del proyecto con una estructura HTML simple
- **styles.css** - Estilos CSS para personalizar la apariencia de la página
- **README.md** - Este archivo con la documentación del proyecto

## Características

- Página web responsiva
- Estilos limpios y modernos
- Estructura HTML semántica

## Propósito

Este proyecto es ideal para:
- Aprender los comandos básicos de Git (commit, push, pull)
- Entender el versionado de código
- Practicar con repositorios locales y remotos
- Familiarizarse con flujos de trabajo colaborativos

## Tecnologías
HTML5, CSS3, ES6

## Instrucciones de uso

1. Clonar el repositorio
2. Abrir el archivo `index.html` en el navegador
3. Explorar el código fuente
